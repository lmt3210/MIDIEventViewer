Version History

1.3.2 (07/05/2025) Added drag and drop support; added selection of drum channel; fixed "open with" support.

1.2.1 (04/19/2025) Changed note name display to be based on key signature; fixed major vs. minor key signature; added registration with macOS to add MIDIEventViewer as an alternate application for opening SMF files; added more meta event support.

1.1.1 (03/16/2025) Fixed parsing of running status; added support for SMPTE, port select, and sequencer specific meta events; added support for sysex events; added event count in header; added resizing of the hex column.

1.0.1 (02/27/2025) Initial release.
